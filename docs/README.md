## Overview

Rainy is a modern, fast weather application built with:

- **Backend:** Node.js + Express
- **Frontend:** Vanilla JavaScript, CSS3
- **APIs:** Open-Meteo (weather), RainViewer (radar), Nominatim (geocoding)
- **Maps:** Leaflet.js

### Key Features

- Real-time weather data (no API key required)
- Interactive radar map with precipitation overlay
- 24-hour and 7-day forecasts
- City search with autocomplete
- Automatic geolocation
- Installable as a PWA (Progressive Web App)
- Fully responsive and accessible design
